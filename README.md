# Recipe Finder

This is a recipe finder app for iOS built with Swift. It provides different ways to discover recipes like searching and uploading receipts. The recipes are retrieved from TheMealDB API.

## Features

- Search for recipes by name
- Upload a receipt of ingredients from your gallery or by taking a photo
- Recipes are randomly shown on the home page for discovery of new recipes
- Favourite recipes to save them
- Scroll through the history of recipes you've seen
- Detailed views of recipes show ingredients and instructions
- Recipes link to Youtube videos that show you how to prepare the meals
